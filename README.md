# Bicycle Shop Sales Analysis Dashboard 

## Project Overview
This is a comprehensive **Power BI Sales Dashboard** created to analyze the business performance of a bicycle retail shop. The project involves transforming raw sales data into interactive visualizations to help stakeholders track key performance indicators (KPIs) and identify growth opportunities.

##  Key Features & Insights
* **Executive KPIs:** Instant visibility into **Total Sales ($118.7M)**, **Total Profit ($16.89M)**, and **Profit Margin (14.23%)**.
* **Geographical Trends:** Waterfall chart showcasing sales distribution by country (Germany, USA, India, etc.).
* **Product Performance:** Identification of top-selling products (e.g., **Paseo**) and units sold percentages.
* **Customer & Segment Analysis:** Breakdown of sales by customer ID and business segments (Government, Small Business, Enterprise).
* **Time-Series Analysis:** Month-over-Month (MoM) sales trend analysis to track seasonal fluctuations.
* **Interactivity:** Dynamic **Year and Month Slicers** for personalized data drilling.

##  Technical Skills Applied
* **Data Source:** Microsoft Excel.
* **ETL Process:** Data cleaning and transformation using **Power Query**.
* **Data Modeling:** Building relationships between tables for accurate reporting.
* **DAX (Data Analysis Expressions):** Created custom measures for Profit %, Total Sales, and Units Sold.
* **UI/UX Design:** Implementing custom theme colors and professional layouts for better readability.
